# Notepad

An open-source simple light-weight notepad.

## Features

- Add Note
- Edit Note
- Delete Note
- Delete Multiple Notes
- Permanently delete note from trash
- Apply tag to Notes
- Add Tag
- Edit Tag
- Delete Tag
- Delete Multiple Tags
